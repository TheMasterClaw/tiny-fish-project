## Slide 8: AI Technology

### Custom TensorFlow.js Model

**Training Data:**
- 10,000+ labeled reef images
- Multiple reef types and locations
- Various lighting conditions

**Capabilities:**
- Coral vs. non-coral segmentation
- Health score regression (0-100%)
- Bleaching classification (none/mild/severe)
- Species detection (20+ coral types)

**Performance:**
- 94% accuracy on test set
- < 5 second inference time
- Runs entirely in browser
- No server required

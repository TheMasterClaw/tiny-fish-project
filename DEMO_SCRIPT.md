## 🚨 Troubleshooting

### "AI analysis taking too long"
- Use smaller image files (<5MB)
- Check internet connection

### "Transaction failed"
- Ensure you have Base Sepolia ETH
- Check gas settings

### "Image upload error"
- Supported formats: JPG, PNG
- Max size: 10MB

---

## 🎥 Recording Tips

1. **Use bright, clear reef images**
2. **Have test images ready** (healthy + degraded)
3. **Pre-load wallet with Base Sepolia ETH**
4. **Zoom to 125%** for better visibility
5. **Emphasize the conservation impact**
